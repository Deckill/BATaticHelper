# 🎯 BA Tactic Helper (블루 아카이브 공략 도우미)

A transparent overlay teleprompter for Blue Archive tactics. 
블루 아카이브 컨텐츠 공략을 화면 위에 띄워두고 편하게 읽을 수 있는 글로벌 단축키 지원 오버레이 프로그램입니다.

### <a id="2-주요-기능"></a> 2. 주요 기능
* **글로벌 단축키:** 게임이 활성화된 상태에서도 Q, E 키로 공략을 이전, 다음으로 넘길 수 있습니다.
* **항상 맨 위에:** 게임이 전체 화면이어도 그 위에 표시됩니다.
* **상단 고정 스크롤:** 지나간 공략과 앞으로 올 공략을 한눈에 볼 수 있도록 현재 순서가 상단에 고정됩니다.
* **투명도 조절:** 게임 화면을 가리지 않도록 배경 투명도를 조절할 수 있습니다.
* **자동 저장:** 작성한 공략과 설정값이 레지스트리에 자동 저장되어, 프로그램을 다시 시작해도 이전의 공략과 설정이 적용됩니다. (총력전때 좋음)

---

### <a id="3-설치-및-실행-방법"></a> 3. 설치 및 실행 방법
1. [Releases](https://github.com/Deckill/BATaticHelper/releases/) 페이지로 이동합니다.
2. 최신 버전의 `BA_Tactic_Helper.exe` 파일을 다운로드합니다.
3. 다운로드한 프로그램을 실행합니다. (단축키 후킹을 위해 관리자 권한이 필요할 수 있습니다.)

---

### <a id="4-상세-사용법"></a> 4. 상세 사용법
1. **공략 입력:** 메모장이나 커뮤니티에서 복사한 공략을 붙여넣기합니다.
2. **공략 수정:** 원하는 부분을 선택해 추가적인 수정을 할 수 있습니다.
3. **단축키 모드 켜기:** 상단의 `단축키 켜기` 버튼을 누르면 텍스트가 잠기며 게임 모드로 돌입합니다.
4. **순서 넘기기:** 단축키(기본 `Q`: 이전, `E`: 다음)를 눌러 공략을 넘기면 됩니다.

![설정 스크린샷](https://github.com/user-attachments/assets/c1a35b56-38fe-49a3-8411-4e63aba96783)
### <a id="5-설정-안내"></a> 5. 설정 안내
상단의 `설정` 버튼을 눌러 다음 항목들을 조절할 수 있습니다.
* **단축키 변경:** Q, E 외에 다른 키나 Shift+Q, Alt+E 등 원하는 조합으로 변경 가능합니다.
* **상단 고정 줄 수:** 현재 순서 '위'에 남겨둘 이전 공략의 줄 수를 설정합니다.
* **투명도 조절 (20% ~ 100%):** 게임 화면이 비치도록 투명도를 조절합니다.
* **색상 및 폰트:** 하이라이트 색상과 글자 크기 변경하실 수 있습니다.

### <a id="2-key-features"></a> 2. Key Features
* **Global Shortcuts:** Even when the game is active, you can navigate to the previous or next step in the guide using the Q and E keys.
* **Always on Top:** It will be displayed on top even if the game is in full screen.
* **Top-Fixed Scrolling:** The current step is fixed at the top so you can see past and upcoming steps at a glance.
* **Transparency Adjustment:** You can adjust the background transparency so it doesn't block the game screen.
* **Auto-Save:** Written guides and settings are automatically saved in the registry, so your previous guide and settings are applied even if you restart the program. (Great for Total Assault)

---

### <a id="3-how-to-install-and-run"></a> 3. How to Install and Run
1. Go to the [Releases](https://github.com/Deckill/BATaticHelper/releases/) page.
2. Download the latest version of the `BA_Tactic_Helper.exe` file.
3. Run the downloaded program. (Administrator privileges may be required for shortcut hooking.)

---

### <a id="4-detailed-usage"></a> 4. Detailed Usage
1. **Enter Guide:** Paste the guide copied from Notepad or a community site.
2. **Edit Guide:** You can select desired parts and make additional edits.
3. **Enable Shortcut Mode:** Click the `Enable Shortcuts` button at the top to lock the text and enter game mode.
4. **Navigate Steps:** Press the shortcuts (Default `Q`: Previous, `E`: Next) to navigate through the guide.

---
![Settings Screenshot](https://github.com/user-attachments/assets/c1a35b56-38fe-49a3-8411-4e63aba96783)
### <a id="5-settings-guide"></a> 5. Settings Guide
Click the `Settings` button at the top to adjust the following items.
* **Change Shortcuts:** You can change it to other keys besides Q and E, or desired combinations like Shift+Q or Alt+E.
* **Number of Top-Fixed Lines:** Set the number of previous guide lines to leave 'above' the current step.
* **Transparency Adjustment (20% ~ 100%):** Adjust the transparency so the game screen is visible.
* **Color and Font:** You can change the highlight color and font size.

### <a id="2-主な機能"></a> 2. 主な機能
* **グローバルショートカット:** ゲームがアクティブな状態でも、Q、Eキーで攻略を前後に進めることができます。
* **常に最前面に表示:** ゲームがフルスクリーンでも、その上に表示されます。
* **上部固定スクロール:** 過ぎた攻略とこれからの攻略を一目で確認できるように、現在の順番が上部に固定されます。
* **透明度調整:** ゲーム画面を隠さないように、背景の透明度を調整できます。
* **自動保存:** 作成した攻略と設定値がレジストリに自動保存され、プログラムを再起動しても以前の攻略と設定が適用されます。（総力戦の時に便利です）

---

### <a id="3-インストールおよび実行方法"></a> 3. インストールおよび実行方法
1. [Releases](https://github.com/Deckill/BATaticHelper/releases/) ページに移動します。
2. 最新バージョンの `BA_Tactic_Helper.exe` ファイルをダウンロードします。
3. ダウンロードしたプログラムを実行します。（ショートカットフッキングのため、管理者権限が必要になる場合があります。）

---

### <a id="4-詳細な使い方"></a> 4. 詳細な使い方
1. **攻略の入力:** メモ帳やコミュニティからコピーした攻略を貼り付けます。
2. **攻略の修正:** 必要な部分を選択して追加の修正を行うことができます。
3. **ショートカットモードをオンにする:** 上部の `ショートカットをオンにする` ボタンを押すとテキストがロックされ、ゲームモードに突入します。
4. **順番を進める:** ショートカット（デフォルト `Q`: 前へ、`E`: 次へ）を押して攻略を進めます。

---
![設定スクリーンショット](https://github.com/user-attachments/assets/c1a35b56-38fe-49a3-8411-4e63aba96783)
### <a id="5-設定の案内"></a> 5. 設定の案内
上部の `設定` ボタンを押して、以下の項目を調整できます。
* **ショートカットの変更:** Q、E以外にも、Shift+Q、Alt+Eなど任意のキーの組み合わせに変更可能です。
* **上部固定行数:** 現在の順番の「上」に残しておく前の攻略の行数を設定します。
* **透明度調整 (20% ~ 100%):** ゲーム画面が透けて見えるように透明度を調整します。
* **色とフォント:** ハイライトの色と文字サイズを変更できます。

### <a id="2-主要功能"></a> 2. 主要功能
* **全局快捷键:** 即使在游戏激活的状态下，也可以使用 Q、E 键来切换上一步或下一步攻略。
* **总在最前面:** 即使游戏是全屏，也会显示在游戏画面之上。
* **顶部固定滚动:** 当前步骤会固定在顶部，以便一眼就能看到已完成和即将到来的攻略步骤。
* **透明度调节:** 可以调节背景透明度，以免遮挡游戏画面。
* **自动保存:** 编写的攻略和设置值会自动保存在注册表中，即使重启程序也会应用之前的攻略和设置。（打总力战时很好用）

---

### <a id="3-安装与运行方法"></a> 3. 安装与运行方法
1. 前往 [Releases](https://github.com/Deckill/BATaticHelper/releases/) 页面。
2. 下载最新版本的 `BA_Tactic_Helper.exe` 文件。
3. 运行下载的程序。（为了实现快捷键拦截，可能需要管理员权限。）

---

### <a id="4-详细使用方法"></a> 4. 详细使用方法
1. **输入攻略:** 粘贴从记事本或社区复制的攻略。
2. **修改攻略:** 可以选择需要的部分进行额外的修改。
3. **开启快捷键模式:** 点击顶部的 `开启快捷键` 按钮，文本将被锁定并进入游戏模式。
4. **切换步骤:** 按下快捷键（默认 `Q`: 上一步，`E`: 下一步）即可切换攻略。

### <a id="5-设置指南"></a> 5. 设置指南
点击顶部的 `设置` 按钮可以调节以下项目。
* **更改快捷键:** 除了 Q、E 之外，还可以更改为其他按键，或使用 Shift+Q、Alt+E 等所需的组合键。
* **顶部固定行数:** 设置保留在当前步骤“上方”的之前攻略的行数。
* **透明度调节 (20% ~ 100%):** 调节透明度以便能看清游戏画面。
* **颜色和字体:** 可以更改高亮颜色和字体大小。
