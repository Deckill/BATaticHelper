# 🎯 BA Tactic Helper (블루 아카이브 택틱 도우미)

A transparent overlay teleprompter for Blue Archive tactics. 
블루 아카이브 총력전/대결전 택틱을 화면 위에 띄워두고 편하게 읽을 수 있는 글로벌 단축키 지원 오버레이 프로그램입니다.

### <a id="2-주요-기능"></a>✨ 2. 주요 기능
* **글로벌 단축키:** 게임이 활성화된 상태에서도 Q, E 키로 택틱을 넘길 수 있습니다.
* **상단 고정 스크롤:** 지나간 택틱과 앞으로 올 택틱을 한눈에 볼 수 있도록 현재 순서가 상단에 고정됩니다.
* **투명도 조절:** 게임 화면을 가리지 않도록 배경 투명도를 실시간으로 조절할 수 있습니다.
* **자동 저장:** 작성한 택틱과 설정값(단축키, 투명도 등)이 레지스트리에 깔끔하게 자동 저장됩니다.
* **다국어 지원:** 시스템 언어를 감지하여 한국어, 영어, 일본어, 중국어(간체)로 자동 표시됩니다.

---

### <a id="3-설치-및-실행-방법"></a>🚀 3. 설치 및 실행 방법
1. [Releases](https://github.com/본인계정/저장소이름/releases) 페이지로 이동합니다.
2. 최신 버전의 `BA_Tactic_Helper.exe` 파일을 다운로드합니다.
3. 다운로드한 프로그램을 실행합니다. (단축키 후킹을 위해 관리자 권한이 필요할 수 있습니다.)

---

### <a id="4-상세-사용법"></a>💻 4. 상세 사용법
1. **택틱 입력:** 메모장이나 커뮤니티에서 복사한 택틱을 프로그램 중앙에 붙여넣기(`Ctrl+V`) 합니다.
2. **이동:** 마우스로 텍스트를 클릭하면 해당 순서로 즉시 이동합니다.
3. **단축키 모드 켜기:** 상단의 `단축키 켜기` 버튼을 누르면 텍스트가 잠기며 게임 모드로 돌입합니다.
4. **순서 넘기기:** 단축키(기본 `Q`: 이전, `E`: 다음)를 눌러 슬롯머신처럼 택틱을 내리며 게임을 플레이합니다.

---

### <a id="5-설정-안내"></a>⚙️ 5. 설정 안내
상단의 `⚙️ 설정` 버튼을 눌러 다음 항목들을 실시간으로 조절할 수 있습니다.
* **단축키 변경:** Q, E 외에 Shift+Q, Alt+E 등 원하는 조합으로 변경 가능
* **상단 고정 줄 수:** 현재 순서 '위'에 남겨둘 이전 공략의 줄 수
* **투명도 조절 (20% ~ 100%):** 게임 화면이 비치도록 조절
* **색상 및 폰트:** 하이라이트 색상과 글자 크기 변경

## <a id="english"></a>🇺🇸 English

A lightweight overlay utility designed to help you easily track complex tactic sequences in Blue Archive using global hotkeys, without obstructing the game screen.

### <a id="1-preview"></a>📺 1. Preview
![App Screenshot](https://via.placeholder.com/600x400?text=Replace+with+your+Image+URL)

### <a id="2-key-features"></a>✨ 2. Key Features
* **Global Hotkeys:** Navigate tactics using hotkeys (Q/E) even while the game is active.
* **Top-Aligned Scroll:** Current step is locked at the top to give you a clear view of upcoming steps.
* **Adjustable Opacity:** Change the background transparency in real-time.
* **Auto-Save:** Tactics and settings are cleanly saved in the Windows Registry.

### <a id="3-how-to-use"></a>💻 3. How to Use
1. Download `BA_Tactic_Helper.exe` from the [Releases](https://github.com/본인계정/저장소이름/releases) page.
2. Paste your tactic text into the app.
3. Click **Enable Hotkeys** to lock the text and activate game mode.
4. Use `Q` (Prev) and `E` (Next) to navigate steps while playing.

*(Settings like Opacity, Custom Hotkeys, and Font Size can be adjusted in the ⚙️ Settings menu!)*
